      Q：

      注意：

      答案中不可以包含重复的四元组。

      示例：

      给定数组 nums = [1, 0, -1, 0, -2, 2]，和 target = 0。

      满足要求的四元组集合为：
      [
      [-1, 0, 0, 1],
      [-2, -1, 1, 2],
      [-2, 0, 0, 2]
      ]
